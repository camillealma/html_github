Historia y revolución de HTML

¿Qué es HTML?

HTML (Hyper Text Markup language) es el lenguage estandar para crear paginas web. Fue desarollado por Tim Berners-lee en 1991.

Evolución de HTML

Desde su creacion, HTML ha pasado por varias versiones importantes que han ampliado sus capacidades y mejorado su funcionalidad.

Versiones Principales de HTML

  • HTML 1.0 (1991)
  • HTML 2.0 (1995)
  • HTML 3.2 (1997)
  • HTML 4.01 (1999)
  • HTML 5 (BORRADOR) (2008)
  • HTML 5 (2014)

Caracterisiticas Destacadas de HTML5

  1. Soporte de multimedia nativa (audio y video).
  2. Elementos semanticos nuevos como <arcticle>, <section >,y < nav>.
  3. Mejor Soporte para graficos y juegos (canvas y SVG).
  4. Almacenamiento web local.
